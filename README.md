# Healthcare-cost-and-Utilization-in-Wisconsin-hospitals
 R-based healthcare analytics project to analyze hospital costs, patient demographics, and length of stay, identifying key cost drivers and supporting efficient resource utilization
 ## Description

This project analyzes healthcare costs and hospital utilization patterns using inpatient data from Wisconsin hospitals (age group 0–17). The objective is to help healthcare agencies understand cost drivers, patient demographics, and resource utilization to support better decision-making.

The analysis focuses on identifying patterns in hospital charges, length of stay (LOS), diagnosis groups, and demographic factors such as age, gender, and race.

## Tech Stack

R Programming – Core analysis

dplyr, tidyverse – Data manipulation

ggplot2 – Data visualization

Statistical Modeling – Relationship analysis & prediction
## Highlights
#### Business Problem


Healthcare agencies need to:

Understand factors driving hospital costs

Identify high-cost patient groups and treatments

Ensure fairness and absence of bias in healthcare charges

Optimize resource allocation and hospital efficiency

#### Goal of the Project

Analyze cost and utilization patterns in hospital data

Identify key factors affecting hospital charges

Study relationships between demographics and healthcare costs

Predict length of stay (LOS) using patient attributes
## Key Analysis Performed
Age-wise Analysis

Identified age groups with highest hospital visits and expenditure

Diagnosis Analysis (APRDRG)

Determined diagnosis groups with maximum hospitalization and cost

Race vs Cost Analysis

Evaluated whether hospital charges vary across different racial groups

Cost Distribution

Analyzed variation of hospital costs across age and gender

Length of Stay Prediction

Built a model to predict LOS using age, gender, and race

Cost Drivers Identification

Identified variables that significantly impact hospital charges (TOTCHG)
## Business Impact & Insights
Identified high-cost patient segments and treatments

Provided insights for better healthcare cost management

Highlighted relationships between demographics and hospital utilization

Supported data-driven resource allocation and planning
## Conclusion

This project demonstrates how R-based data analysis can uncover meaningful insights from healthcare data, helping organizations improve cost efficiency, fairness, and operational decision-making.
